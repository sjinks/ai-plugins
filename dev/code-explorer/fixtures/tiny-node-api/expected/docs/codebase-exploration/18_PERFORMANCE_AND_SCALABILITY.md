### PERF-001: Unbounded reflected response size

- File: `src/routes.js` (`handleGreeting`)
- The `name` query parameter is reflected into the response with no length bound, so response size is attacker-controlled.
- Why it matters: a very long `name` produces a proportionally large response and allocation.
- Suggested benchmark: measure response time and memory for `name` lengths of 1KB, 1MB, and 10MB.
