# Expected Report Shape

- Coordinator normalizes the raw diff as supplied context.
- Contextual, independent, security, performance, adversarial, and test-adequacy lenses are selected or limitations are stated.
- Security lens can report an evidence-anchored SQL injection finding.
- Final report includes `Review Status`, `Inputs`, `Lenses Run`, `Findings`, `Cross-Lens Conflicts`, `Recommendation`, `Limitations`, `Residual Risk`, and `Deferred`.
